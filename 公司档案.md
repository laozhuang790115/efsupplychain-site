## 六、五大业务流程

### 1. 代采代理流程

```
原料工厂/品牌供应方 → 以鲜集采（源头集采·一手货源）
    → 以鲜集采·乾联鲜行平台（供应链+共享+大数据+可视服务）
    → 第三方服务生态（清关·加工·保险·仓储·金融·冷链运输）
```

- 品牌供应方：Santa Priscila · Mowi

### 2. 贸易融资 — 进口供应链全流程

```
经销商（20%订金） → 大贸易商 → 资金方（融资80%）
    → 海外采购（CIF方式，TT 20%订金）
    → 国际物流（25-60天海运，到港前7天付尾款换提单）
    → 通关入仓（报关·报检·清关·检验检疫·入库）
    → 还款提货（资金方放货指令，还清尾款，分批提货）
```

- **四流合一 SaaS 平台**：合同付款 / 航运物流 / 报关清关 / 运输追踪 / 仓储管理

### 3. 报关报检流程

```
货物到港 → 单证准备 → 以鲜报关团队（预审单证·归类编码）
    → 电子申报·单一窗口（海关审核·查验放行·缴税）
    → 检验检疫（动物疫病·兽药残留·标签合规）
    → 入境检验检疫证 → 海关放行 → 提货出闸
```

- 时效：普通单证 1-2 工作日放行
- 覆盖口岸：深圳·广州·湛江·上海·宁波·天津·青岛
- 合规：零退运·零处罚记录

### 4. 冷链仓储物流全流程

```
入库验收 → 温控存储（-18°C冷冻 / 0-4°C冷藏，24h监控）
    → 库存管理（WMS，批次/效期/库龄预警）
    → 订单履约（接单·拣货·复核，支持分批出货·一件代发）
    → 冷链配送（冷藏车城市配送·干线冷链运输，全程温度可追溯）
```

- 仓储节点：9大城市
- 温区：冷冻库 -18~-25°C / 冷藏库 0-4°C / 恒温库 15-18°C
- 增值服务：换标贴标·分装打包·质检抽检·库存质押
- 配送：城市配送当日达·干线冷链次日达·全国门到门

### 5. 清真食品出海全流程

```
产品评估（原料溯源·配方审核·生产线合规）
    → HALAL认证（MUI·JAKIM·MUIS·GAC）
    → 好品中国出海平台（海外展会·国家食品馆·渠道撮合）
    → 海外落地（出口报关·国际冷链物流·目的国标签合规·海外仓配·渠道入驻）
```

- 展会资源：MIFB马来西亚 · Gulfood迪拜 · Thaifex泰国 · SIAL法国 · ANUGA
- 认证覆盖：MUI · JAKIM · MUIS · GAC · FDA
